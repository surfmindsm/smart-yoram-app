# Smart Yoram 앱 개인정보처리방침 및 Google Play Console 설정 가이드

## 1. 개요
이 문서는 Smart Yoram 앱의 CAMERA 권한 사용에 대한 개인정보처리방침 준수 및 Google Play Console 설정 방법을 안내합니다.

## 2. 구현된 개인정보 보호 조치

### 2.1 Android 매니페스트 설정
- **권한 명시**: `android.permission.CAMERA`, `android.permission.FLASHLIGHT`
- **개인정보처리방침 메타데이터**: 권한 사용 목적 및 정책 안내
- **연락처 정보**: 개인정보 문의 이메일 주소

### 2.2 앱 내 개인정보처리방침
- **전체 개인정보처리방침 페이지**: `lib/screens/privacy_policy_screen.dart`
- **설정 메뉴 접근**: 설정 > 개인정보 처리방침
- **권한 요청 시 안내**: 카메라 권한 요청 시 상세 설명 제공

### 2.3 권한 관리 시스템
- **사전 동의**: 권한 사용 전 개인정보처리방침 동의 요구
- **투명성**: 권한 사용 목적 명확히 고지
- **선택권**: 권한 거부 시에도 앱 기본 기능 사용 가능

## 3. Google Play Console 설정 가이드

### 3.1 앱 콘텐츠 설정
1. **Google Play Console** 로그인
2. **앱 선택** > **정책** > **앱 콘텐츠**
3. **개인정보처리방침** 섹션에서 다음 정보 입력:
   - 개인정보처리방침 URL: `https://smartyoram.com/privacy-policy`
   - 연락처 이메일: `privacy@smartyoram.com`

### 3.2 데이터 보안 섹션
1. **데이터 보안** 메뉴로 이동
2. **카메라** 권한에 대해 다음과 같이 설정:

#### 카메라 액세스
- **데이터 수집 여부**: 예
- **수집 목적**:
  - ☑️ 앱 기능 제공
  - ☑️ 분석 목적
  - ☑️ 개발자 커뮤니케이션
- **데이터 공유**: 아니요 (제3자와 공유하지 않음)
- **데이터 전송**: 예 (암호화하여 전송)
- **사용자 제어**: 예 (사용자가 데이터 삭제 요청 가능)

#### 설명 텍스트
```
Smart Yoram 앱은 다음 목적으로만 카메라를 사용합니다:
• QR 코드 스캔을 통한 출석 체크
• 교회 행사 및 학습 활동 사진 촬영  
• 교인 프로필 사진 등록

모든 촬영 데이터는 사용자 동의 하에서만 수집되며, 교회 관리 목적으로만 사용됩니다. 수집된 개인데이터는 암호화되어 안전하게 저장되며, 사용자의 동의 없이 제3자와 공유되지 않습니다.
```

### 3.3 권한 설명
**android.permission.CAMERA** 권한에 대한 설명:
```
이 권한은 QR 코드 스캔을 통한 교회 출석 체크, 교회 행사 기록용 사진 촬영, 교인 프로필 사진 등록을 위해 사용됩니다. 촬영된 사진은 사용자의 명시적 동의 하에서만 저장되며, 교회 관리 및 서비스 제공 목적으로만 사용됩니다.
```
